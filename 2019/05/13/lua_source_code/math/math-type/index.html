math_type简介

math_type是lua库函数math.type的具体实现。math_type的作用是判断一个参数是整数还是浮点数：

  • 如果参数是整数，则返回“integer”；
  • 如果参数是浮点数，则返回“float”；
  • 如果参数不是数字，则返回nil。
