math_randomseed通过调用l_srand将传入的参数设为随机数生成器的“种子”。注意接下来的(void)l_rand();，执行一次随机数函数的调用的目的是丢弃当前“种子”产生的第一个随机数，避免产生不好的随机数关联，以便提高伪随机性。
